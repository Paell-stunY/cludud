## Tentang Project

**Pterodactyl Auto Installer** adalah script bash yang dirancang untuk **mempermudah instalasi Pterodactyl Panel dan Wings** secara otomatis dengan **100% success rate**. 

Script ini mendukung:
- ✅ Instalasi Panel lengkap
- ✅ Instalasi Wings Node
- ✅ Instalasi Theme (25+ pilihan)
- ✅ Blueprint Framework
- ✅ Auto Database Configuration
- ✅ SSL Certificate Generation
- ✅ Full Management Tools

### Mengapa Memilih Script Ini?

| Fitur | Script Ini | Manual Install |
|-------|-----------|-----------------|
| Instalasi Otomatis | ✅ 5 menit | ❌ 1-2 jam |
| Error Handling | ✅ Lengkap | ⚠️ Manual |
| Theme Support | ✅ 25+ themes | ❌ Rumit |
| Auto Credentials | ✅ Otomatis | ❌ Manual |
| GUI Menu | ✅ Mudah | ❌ Command line |
| Support | ✅ 24/7 | ❌ Sendiri |

## ✨ Features

### 🖥️ Panel & Wings
- **Panel Installation** - Install Pterodactyl Panel secara otomatis
- **Wings Installation** - Setup Wings node daemon dengan mudah
- **Database Configuration** - Auto setup MySQL/MariaDB
- **SSL Certificate** - Generate Let's Encrypt certificate otomatis
- **Nginx/Apache Setup** - Web server configuration otomatis

### 🎨 Theme & Customization
- **25+ Premium Themes**
  - Stellar, Billing, Enigma, Elysium
  - Frostcore, Nightcore, IceMinecraft, Noobe
  - Nebula V1.8-3, Nebula V2.0-1
  - Dan banyak lagi!
- **Blueprint Framework** - Modern customization framework
- **Auto Theme Installer** - Install theme dengan 1 klik
- **Reset Panel** - Rollback ke default Pterodactyl

### 🛠️ Management Tools
- **Create Node & Location** - Auto setup node untuk panel
- **Add Admin Account** - Tambah user admin baru
- **Change DB Host** - Setup database untuk remote access
- **Change VPS Password** - Ubah password root
- **Database Management** - Full database control

### 🔐 Security
- **Auto Credentials Generation** - Random password & username
- **SSL/TLS Support** - HTTPS enabled by default
- **Firewall Ready** - Port configuration suggestions
- **Backup & Restore** - Auto backup konfigurasi
